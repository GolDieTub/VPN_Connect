# VPN Connect

Проект представляет собой переработанный интерфейс VPN-клиента на основе исходного макета с улучшением визуальной иерархии, пользовательского сценария и поведения интерфейса.

## Как запустить проект

1. Склонировать репозиторий через команду в терминале "git clone https://github.com/GolDieTub/VPN_Connect.git" или скачать архив и разархивировать его
2. Открыть проект в Xcode
3. Нажать Run или комбинацию Cmnd + R

### Поведение при смене сервера во время активного подключения

Если пользователь выбирает новый сервер в состоянии `Connected`, приложение выполняет последовательность:
- `Disconnecting` от предыдущего сервера
- смена выбранной локации
- `Connecting` к новому серверу
- `Connected`

Таким образом, смена сервера моделирует реалистичный сценарий переподключения.

## Что было улучшено относительно исходного дизайна

Исходный макет содержал несколько проблем:
- слабая визуальная иерархия
- перегруженный акцент на огромной круглой кнопке
- неочевидная структура экрана
- второстепенные действия размещены не по паттернам macOS
- декоративная карта не была связана с выбранной страной
- элементы плохо работали при изменении размеров окна
- добавлена верхняя панель macOS
- обработан сценарий горячей смены смены сервера

### Основные доработки интерфейса

#### 1. Перестроена визуальная иерархия
Интерфейс был разбит на логические блоки:
- карта мира
- статус подключения
- список серверов
- основное действие подключения
- дополнительная информация о состоянии соединения

#### 2. Главный action сделан более нативным для macOS
Вместо визуально перегруженного старого решения главный сценарий подключения оформлен как отдельный primary action с понятным расположением и акцентом.

#### 3. Улучшен статусный блок
Статус подключения теперь отображается как отдельная карточка с:
- текущим состоянием
- поясняющим текстом
- индикатором активности
- визуальным акцентом через системные иконки и цвет

#### 4. Переделан выбор сервера
Список стран реализован через `ScrollView`, что позволяет:
- удобно масштабировать список
- легко добавлять новые серверы
- поддерживать большее количество локаций без изменения базовой структуры экрана

#### 5. Добавлена карта мира с привязкой к координатам
Карта отображает выбранный сервер в виде точки по координатам `latitude` / `longitude`.

Это позволяет:
- автоматически отображать новые серверы при добавлении в список
- не рассчитывать позиции вручную
- связать выбор сервера с визуальным отображением на карте

#### 6. Карта адаптируется к теме macOS
Карта оформлена в монохромном стиле и меняет внешний вид в зависимости от светлой или тёмной темы.

#### 7. Исправлено масштабирование окна
Интерфейс учитывает минимально допустимый размер окна, чтобы:
- все блоки всегда помещались
- сохранялись внешние и внутренние отступы
- список серверов не ломал layout
- интерфейс выглядел предсказуемо при ресайзе

#### 8. Добавлен таймер активного соединения
После подключения под кнопкой отображается время активной сессии.

Таймер:
- запускается при переходе в `Connected`
- сбрасывается при отключении
- работает на общем состоянии приложения

#### 9. Добавлено управление через Menu Bar Extra
Приложение поддерживает управление из верхней панели macOS, аналогично desktop VPN-клиентам.

Из Menu Bar доступны:
- просмотр текущего статуса
- просмотр выбранного сервера
- выбор сервера
- подключение и отключение
- открытие основного окна

#### 10. Цветовая синхронизация элементов
В зависимости от сосотояния подключения, цвет определнных элементов интерфеса меняется

#### 11. Состояния синхронизированы между окном и Menu Bar
Окно приложения и Menu Bar Extra используют единый источник состояния.

Это означает, что:
- изменение сервера в окне сразу отражается в верхней панели
- подключение из Menu Bar сразу меняет состояние в основном интерфейсе
- таймер и статус остаются синхронизированными в обоих местах

## UX-решения

В проекте были приняты следующие UX-решения:

- основной пользовательский сценарий сфокусирован вокруг одного главного действия
- второстепенные действия вынесены в более ожидаемые для macOS места
- статус подключения всегда виден и легко считывается
- список серверов остаётся в явной зоне внимания
- переподключение при смене сервера ведёт себя последовательно и предсказуемо
- приложение остаётся удобным как в обычном окне, так и через Menu Bar
- интерфейс не разваливается на маленьких размерах окна

## Поддержка расширения

Текущая структура позволяет без существенной переработки:
- добавлять новые страны и серверы
- расширять список состояний
- интегрировать реальные VPN-события вместо stub-логики
- сохранять пользовательские настройки
- добавлять дополнительные параметры подключения
- развивать карту и статусный UI
